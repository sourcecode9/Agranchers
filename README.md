# Agranchers LLP Website

A lightweight, modern static website for Agranchers LLP focused on premium Rhodes Grass production in Pakistan.

## What is included

- Hero section
- Premium Rhodes Grass producer positioning
- 4,000+ acre production scale messaging
- Clickable project snapshot section with galleries and video placeholders
- Quality, yield, and moisture control sections
- Technical expertise section
- Team section
- Contact us section
- English/Arabic language switcher

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Notes

- Replace the placeholder YouTube IDs in `main.js` with your real project video IDs to turn the placeholders into live embeds.
- Replace the generated Rhodes Grass images in `assets/` with your actual field photos when available.
- Replace the placeholder contact details in `index.html` with your actual business information.
- Update the English and Arabic copy in the `translations` object in `main.js`.
